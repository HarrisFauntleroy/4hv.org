# Rails

## Getting Started

Install the required gems by running the following command:

```bash
bundle install
```

To migrate the database, run the following command:

```bash
rails db:migrate
```

To run the test suite, run the following command:

```bash
rails test
```

To start the server, run the following command:

```bash
rails server
```
